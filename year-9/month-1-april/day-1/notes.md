ACTIVE RECALL QUESTIONS:

1. What is the purpose of <!DOCTYPE html> and how does it influence how a browser displays a webpage?
  <!DOCTYPE html> is used to define hte html version to the web browser

2. Which types of elements belong in the <head> section versus the <body>, and why is <title> placed in <head>?
  The <head> section is used to store stuff meant for the website that the user doesn't have to see such as the language or what it's encoded in (UTF-8..) while the <body> is used to create actual elements the user sees on 
  the webpage

3. What is the difference between <ul> and <ol>, and in what situations would you use each one?
  <ul> means Unordered List and is pretty much a bullet-pointed list for stuff that don't have to be in order while <ol> means Ordered List and is a numbered list starting at 1. used to give instructions in order
  
4. What do <em> and <strong> represent in HTML, and what happens when you nest them inside another element like <li>?
  <em> is used to put the nested word in Italics and it means Emphasis while <strong> is used to put the nested word in Bold. If you were to use one of these inside of an element, still, only the nested word gets affected
